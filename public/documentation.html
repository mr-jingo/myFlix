Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie,
Example:
{
id: 1,
title: 'Conan the Barbarian',
director: 'John Milius'
}
Get a list of users /users GET None A JSON object holding data about all the users
Get data about a single user by name /users/[name] GET None A JSON object holding data about a single user,
Example:
{
name: 'user1',
password: 'user1!'',
mail: 'user1@gmx.net'
}
Add an user /users POST A JSON object holding data about the user to add,
Example:
{
name: 'user1',
password: 'user1!'',
mail: 'user1@gmx.net'
} 		A JSON object holding data about the user to add,
Example:
{
name: 'user1',
password: 'user1!'',
mail: 'user1@gmx.net'
favoriteMovie: [],
__v: 0
}
Add a movie by ID to the favorite list of an user /users/[name]/movies/[movieId] POST None A JSON file with MovieID as vlaue of FavoriteMovies key
Update a certain user /users/[name] PUT None A text message indicating wheter the user was updated
Delete an user /users/[name] DELETE None A text message indicating whether the user was successfully removed
Remove a movie by ID from the FavoriteMovies of an user /users/[name]/movies/[movieId] DELETE None A text message indicating whether the movie was successfully removed
